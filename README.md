# GoTogether Mobile 🚗📱

GoTogether é uma plataforma de mobilidade compartilhada desenvolvida como Trabalho de Conclusão de Curso (TCC), criada com o objetivo de conectar motoristas e passageiros que possuem trajetos em comum, permitindo organizar caronas de forma prática, segura e eficiente.

O projeto busca oferecer uma alternativa mais acessível de deslocamento, promovendo economia, conveniência e interação social entre usuários.

---

## ✨ Funcionalidades

### 🔐 Sistema de autenticação
- Cadastro de usuários
- Login seguro
- Separação entre perfis de motorista e passageiro

### 🚘 Sistema de caronas
- Busca e seleção de destinos
- Definição de rotas
- Cadastro de viagens

### 📍 Geolocalização
- Integração com mapas interativos
- Busca inteligente de destinos
- Sugestões automáticas de localização

### 👤 Gerenciamento de perfil
- Edição de informações pessoais
- Preferências de viagem
- Configurações específicas para motoristas e passageiros

### 🎨 Interface
- Navegação por Drawer Navigation
- Bottom Sheet interativo
- Feedback visual com mensagens dinâmicas
- Interface responsiva para mobile e web

## 🚀 Executando o projeto

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/gotogether-mobile.git
```

Instale as dependências:

```bash
npm install
```

Inicie o projeto:

```bash
npx expo start
```

Executar no Android:

```bash
npm run android
```

Executar no navegador:

```bash
npm run web
```
